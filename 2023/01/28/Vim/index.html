tips

Basic

1.命令模式
2.输入模式
3.底线命令模式
:wq save and quit
:w filename ave as other file
:set nu line number
:set nonu

i/I 输入字符
a/A 输入字符
o/O 新起一行
r/R 替换字符

移动光标操作

移动光标 hjkl/⬅️⬇️⬆️➡️
G 移动到最后一行
gg 移动到第一行
0 移动到行首
$ 移动到行尾

nG n为数字，移动到第n行（配合:set nu显示行号）
n n为数字，光标向下移动n行
ctr+f page down
ctr+b page up
ctr+d half page down
ctr+u half page up

文本操作

dd 删除当前行
ndd n为数字，删除n行
x delete
X Backspace
nx/nX 连续删除n个字符
yy 复制当前行
nyy n为数字，复制n行
p/P 粘贴复制的内容
v/V/Ctr+v 可视化模式
/word 搜索关键词，n下N上，从当前光标往下搜索（:set hlsearch 高亮显示）
?word 搜索关键词，从当前光标往上搜索
u,Ctr+r u撤销，ctr+r重做
:n1,n2s/wd1/wd2/g 搜索替换关键词 【eg. :1,$s/wd1/wd2/g :1,$s/wd1/wd2/gc】
